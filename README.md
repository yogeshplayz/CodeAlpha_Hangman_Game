# CodeAlpha_Hangman_Game

## Overview
A classic Hangman game implementation created as part of the **CodeAlpha Internship Program**. This project demonstrates core programming concepts including game logic, user input handling, and interactive command-line gameplay.

## Description
Hangman is a timeless word-guessing game where players attempt to guess a hidden word by suggesting letters one at a time. For each incorrect guess, a part of the hangman drawing is added. The challenge is to guess the word before the drawing is completed.

### Features
- **Interactive Gameplay**: Guess letters to reveal the hidden word
- **Score Tracking**: Monitor correct and incorrect guesses
- **Word Selection**: Random word selection from a predefined word list
- **Visual Feedback**: Display of guessed letters and remaining attempts
- **Game States**: Win/lose conditions with appropriate game endings
- **User-Friendly Interface**: Clear prompts and intuitive controls

## How to Play
1. Run the game
2. A random word is selected and hidden (shown as dashes)
3. Guess one letter at a time
4. Correct guesses reveal the letter's position(s) in the word
5. Incorrect guesses add to the hangman drawing
6. Win by guessing the word before running out of attempts
7. Lose when the hangman drawing is complete
